# **App Name**: TaskFlow

## Core Features:

- Task Creation: Allow users to add new tasks to their list.
- Task Editing: Enable users to edit existing tasks to update descriptions or details.
- Task Deletion: Allow users to remove tasks from their list.
- Task Completion: Enable users to mark tasks as complete, with a visual indication.
- Checklist Support: Tasks can behave like checklist items, supporting multiple sub-tasks.
- AI-Powered Task Suggestions: Suggest related tasks based on current entries using a generative AI tool.
- Theme Switching: Allow users to switch between a light and dark theme for the application.

## Style Guidelines:

- Primary color (Light Theme): Soft blue (#64B5F6) to convey calmness and focus.
- Background color (Light Theme): Very light blue (#E3F2FD), almost white, to ensure readability and a clean interface.
- Accent color (Light Theme): Light indigo (#7986CB) to provide a gentle highlight for interactive elements.
- Primary color (Dark Theme): Dark grey (#303030) for a comfortable viewing experience in low-light conditions.
- Background color (Dark Theme): Black (#000000) to minimize eye strain.
- Accent color (Dark Theme): Teal (#26A69A) to provide a gentle highlight for interactive elements.
- Body and headline font: 'PT Sans' for a modern, clear, and readable experience in both themes.
- Use simple, line-based icons to represent different task types and actions. Adapt icon colors for optimal visibility in each theme.
- A minimalist layout with ample whitespace for a distraction-free experience in both themes.
- Subtle transitions and animations to provide feedback on user interactions. Adjust animation speed and intensity based on the selected theme.
